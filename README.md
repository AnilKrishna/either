## Either
Either pair class for the Rx response usage

### Install

Add jitpack to your root `build.gradle`
```
	allprojects {
		repositories {
			...
			maven { url "https://jitpack.io" }
		}
	}
```
	
Add the dependency
```
	dependencies {
	        compile 'com.github.zetaapps:either:0.3'
	}
```
